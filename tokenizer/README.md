# Disclaimer
This tokenizer is based on the speechbrain recipe "LibriSpeech" tokenizer by Mirco Ravanelli.

This folder contains the scripts to train a tokenizer using SentencePiece (https://github.com/google/sentencepiece).
The tokenizer should be trained on the training transcriptions.

# How to run

Create a train, test, dev, set. Train tokenizer.

```bash
sh run.sh
```
